Charboxes- Character Centric Structured Information

These are the visualisations of a few interesting books, as obtained from the Charboxes System. The Charboxes System has been accepted at ACM SIGIR 2014 Gold Coast, Australia.

The main source code for the system is available on request from the authors - pbansal@linkedin.com. The visualisations use an awesome javascript library called D3.
